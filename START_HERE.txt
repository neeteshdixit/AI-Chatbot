================================================================================
  🐛 TROUBLESHOOTING
================================================================================

Models not loading?
  → Check: dir models\emotion_detector
  → Check: dir models\response_model
  → If missing, run: python train_emotion.py && python fine_tune.py

Port already in use?
  → Find: netstat -ano | findstr :8000
  → Kill: taskkill /PID <PID> /F

Frontend won't start?
  → cd frontend
  → npm install
  → npm run dev

Backend errors?
  → Check terminal output
  → Ensure models are trained
  → Try: python app.py
